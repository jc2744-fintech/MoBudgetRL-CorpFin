# Architecture

- `morlba.envs.budget_env`: stochastic returns with synergies, fairness & strategy metrics
- `morlba.agents.pg`: scalarization-based policy gradient
- `morlba.agents.evo`: simple Pareto sampler (evolutionary baseline)
- `morlba.eval.frontier`: compute estimated Pareto frontier & summaries
- `morlba.cli`: Typer CLI
